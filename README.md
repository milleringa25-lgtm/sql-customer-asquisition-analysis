## Аналіз залучення клієнтів та доходу (SQL Portfolio Project).

## Огляд проєкту
Цей проєкт присвячений аналізу каналів залучення клієнтів для інтернет-магазину з використанням **SQL** та **SQLite**. 

Основна мета — оцінити ефективність маркетингових каналів та визначити, які джерела приносять найбільший обсяг клієнтів, генерують найвищий дохід, стимулюють повторні покупки та забезпечують найкращий прибуток у розрахунку на одного клієнта.

---

## Бізнес-питання
**Які канали залучення приносять найбільш цінних клієнтів?**

### Додаткові дослідницькі питання:
* Які канали залучення найефективніші за виручкою серед жінок (TOP-5 каналів)

---

## Інструменти та технології
* **СУБД:** SQLite
* **Мова запитів:** SQL
* **SQL-клієнт:** DBeaver
* **Джерела даних:** Імпорт CSV-файлів

## Набір даних та схема бази даних
Проєкт базується на двох основних датасетах, імпортованих із файлів CSV у таблиці SQLite:

| Файл / Таблиця | Опис |
| :--- | :--- |
| `customers` | **Профілі клієнтів:** ID користувача, стать, вік, місто, канал залучення та дата реєстрації. |
| `orders` | **Деталі замовлень:** ID замовлення, дата замовлення, сума покупки, канал оформлення, спосіб оплати, дні доставки та статус замовлення. |

## Етапи аналізу
1. **Аудит даних:** Перевірено завантажені дані, часові діапазони та загальні суми доходу.
2. **Налаштування бази даних:** Створено схему бази даних SQLite та налаштовано таблиці `customers` і `orders`.
3. **Об'єднання даних:** Об'єднано датасети клієнтів та замовлень за допомогою поля `user_id`.
4. **Розрахунок основних метрик ефективності каналів:** Розраховано загальну кількість клієнтів, кількість замовлень, сукупний дохід та середній чек з групуванням за каналами залучення та TOP-5 каналів серед жінок.
5. **Аналіз цінності користувачів:** Обчислено дохід на одного клієнта для оцінки їхньої фінансової цінності.
6. **Аналіз утримання (Retention):** Досліджено повторні покупки та лояльність клієнтів у розрізі каналів.
7. **Аналіз утримання (Retention Rate):** Досліджено повторні покупки та лояльність клієнтів у розрізі каналів у відсотках.
8. **Аналіз трендів:** Вивчено щомісячну динаміку залучення клієнтів та надходження доходу.
