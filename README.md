# Customer Acquisition Channels Analysis

## Project Overview
Analyzes customer acquisition channels to identify which channels bring the most valuable customers.

## Business Question
Which acquisition channels bring the most valuable customers?

## Tools
- SQL
- SQLite
- DBeaver

## Analysis Steps
- Imported CSV data into SQLite.
- Cleaned and prepared tables for analysis.

## Key Insights
- Channel A generated the highest total revenue.
- Channel B had the highest revenue per customer.
- Some channels brought many customers but lower average customer value.
